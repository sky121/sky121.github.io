Class Alert

A watchful Python script that refreshes the page so you never have to — and writes the moment a seat opens.

During my undergraduate studies at the University of California, Irvine, course enrollment brought a familiar litany of frustrations: bad wifi, server crashes, full courses, and stressful waitlists. During one especially bad episode, with the enrollment site crashing again and again, I spent hours refreshing a single page in hopes of claiming a spot in an important course. After so much time lost to so trivial a task, I decided there had to be a better way — an application that would watch the page on my behalf. I chose a Python web scraper to load and refresh the page and detect the instant my course opened.

I called the project Class Alert. Built in Python, it sends an alert by email and text the moment a course becomes available. When the script runs, it takes the department and the course code of the class you want, reports in real time whether the class is open, and lets you set both the refresh interval and the contact details to ping when a seat appears.

I found I was deeply engaged by this project precisely because it solved a frustration I was living through — hours would pass without my noticing. Though the work was not highly technical, it opened a whole new realm of web scraping for me, and it was a gratifying way to carry the fundamentals from my coursework into the real world.
